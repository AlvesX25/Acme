Elementos de Listas de descrição

TCP
Trasmission Control é um protocolo usado junto o protocolo de internetr (IP) para enviar dados na formas de unidades de mensagens, chamadas de pacotes, entre computadores de internet.
IP
Internet Protocol is um Protocolo pelo qual dados são enviados de um computador para outro na internet. Cada computador na internet é indetificado poe endereço de Ip único.
FTP
File Trnasfer Protocol é um protocolo usado para trocas de arquivos entre computador na Internet.
HTTP
Hypertext Transfer Protocol é um protocolo usado para trocas de arquivos de texto, imagens gráficas, som,vídeo e outros tipos de multimidias de rede.
